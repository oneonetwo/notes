## 三. 设计原则 SOLID

        - S 单一职责原则
        - O 开放封闭原则
        - L 里氏替换原则
        - D 依赖倒置原则: 依赖抽象并非具体的实现。
        - I 接口隔离原则：保持接口的单一独立，避免实现胖接口。

    3. 合成复用原则
