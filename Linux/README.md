### Linux文件和目录管理

3. echo 打印
    - 写入的作用就是这个">"在linux中叫做重定向，即把前面生产的输出写入到后面的文件中
    - `>` 是覆盖 `>>` 是追加
4. mkdir `mkdir[-mp][目录名称]` 
    - `-m` 设置目录权限 `-p` 递归创建目录
5. rm `rm[-rf][目录]` 
    - `-r`可以删除不为空的目录  `-f`强制删除  `-i`提示是否删除    
6. alias
    - 用来设置指令的别名语法： `alias[别名]=[指令名称]`  例如 `alias rm='rm -i'`
